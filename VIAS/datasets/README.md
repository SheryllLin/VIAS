# Dataset Layout

Place the required datasets in these local directories:

- `datasets/CrowdHuman`
- `datasets/COCO2017`
- `datasets/MOT17`
- `datasets/LFW`
- `datasets/Market1501`
- `datasets/PRW`
- `datasets/NTU_RGBD_120`
- `datasets/CCTV_Action`

These folders are intentionally local-only. No cloud download logic is included.
